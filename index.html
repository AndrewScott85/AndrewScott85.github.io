About Me

I'm currently studying full-stack web development at iO-Academy in Bath. Prior to joining this course, I'd worked in several industries, most recently working as a scuba diving instructor.

After deciding it was time for a career change, I remembered how much I'd enjoyed the introductory coding courses completed at university and decided now is the time to follow this up.

Portfolio

Below is a selection of projects showcasing some of the work I've done.

An OOP application in PHP, styled using SASS, which takes data on a collection of cars from an API in JSON format, uses it to build a database in MySQL, then fetches that data to display a searchable, filterable collection of cars - clicking on any of the cars opens a new page showing car information in more detail. This was a great project for getting into working in an OOP manner, as well as good practice in SQL.

The front end of a food delivery app, built in React and styled using Bootstrap, to a specification laid out in high fidelity wireframes. Fetching from an API to display a list of available restaurants and a filterable menu for each restaurant. Items can be added an order basket, which calculates the total cost. Submitting an order opens a modal window displaying the ETA and a progress bar. This app provided a good chance to get familiar with working in React and Bootstrap to produce an App to match a given visual design.

A word guessing game built using javascript. Target words and synonyms are fetched from an api, and on starting a new game, a random target word is selected, and its synonyms are displayed to the player, who is given 5 attempts, or a maximum of 30 seconds to guess the target word. This app was really fun to make, and good practice in javascript and css.
